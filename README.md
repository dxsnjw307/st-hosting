## Das Prepaid-Modell – und warum das cleverer ist als es klingt

Das klingt erstmal ungewohnt: Du lädst Guthaben auf, der Server zieht monatlich davon ab. Kein Lastschriftmandat, keine automatische Verlängerung ohne Warnung. Wenn das Guthaben alle ist und du nicht nachladest, werden deine Daten noch ein paar Tage gespeichert – genug Zeit, um zu reagieren.

Zahlen kann man per PayPal, Sofortüberweisung, Barzahlen, PaySafeCard oder klassische Banküberweisung. Das ist auch für Nutzer interessant, die kein Girokonto haben oder bewusst anonym zahlen wollen.
